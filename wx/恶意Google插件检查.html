恶意Google插件检查

这条链接列出了恶意Google插件的id(信源在这里)。这里提供一个快速检查是否中招的办法。

  1. 首先，在Chrome中打开 chrome://extensions；
  2. 在刚才打开的页面中打开Chrome控制台并运行以下脚本：
    copy(
      "ids=" +
        JSON.stringify(
          Array.from(
            document
              .querySelector("extensions-manager")
              .shadowRoot.querySelector("cr-view-manager")
              .querySelector("extensions-item-list")
              .shadowRoot.querySelectorAll("extensions-item"),
            (v) => v.id
          )
        )
    );
    脚本的作用是复制你安装的所有插件的id并赋值给变量ids
  3. 打开恶意插件页；
  4. 打开恶意插件页的Chrome控制台，粘贴第二步中复制的数据到控制台并回车；
  5. 继续在恶意插件页的控制台追加以下内容并回车
    document.body.textContent.split("\n")
      .filter((v) => ids.includes(v));

以上操作完成后，如果返回的是个空数组，则没中招；否则就中招了，数组中的元素是中招插件的id。
